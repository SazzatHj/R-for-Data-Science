# R for Data Science: My Learning Journey & Analytics Notebook

Welcome to my open-source learning repository for Data Science and Statistical Research using R! This repository serves as a comprehensive digital notebook documenting my journey from foundational concepts to advanced data analysis and visualization.

---

## Repository Purpose & Intent
* **Structured Learning:** To build a bulletproof foundation in R Programming and Data Science workflows.
* **Research Readiness:** Preparing portfolio-ready components for academic research, geopolitics analysis, and PhD applications.
* **Knowledge Sharing:** Acting as a step-by-step guide for student initiatives, researchers, and beginner data scientists who want to learn R seamlessly.

## How to Use This Repository

### ১. নতুন শিক্ষার্থীদের জন্য (For Learners):
আপনি যদি একদম শুরু থেকে আর (R) শিখতে চান, তবে `01-R-Basics` ফোল্ডার থেকে শুরু করুন। প্রতিটি ফাইলে কোডের পাশাপাশি বাংলায় বিস্তারিত ব্যাখ্যা (Detailed Explanations) দেওয়া আছে, যা আপনার বুঝতে সুবিধা হবে।

### ২. কোড প্র্যাকটিস করার নিয়ম (How to Run the Code):
1. এই রেপোটি ক্লোন বা জিপ ফাইল হিসেবে ডাউনলোড করুন।
2. আপনার পিসিতে **R** এবং **RStudio** ইনস্টল করা থাকতে হবে।
3. যেকোনো `.Rmd` বা `.md` ফাইলের কোড ব্লক (` ```r `) কপি করে আপনার RStudio কনসোলে রান করুন।
